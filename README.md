# Tsubu 粒粒 — Legal &amp; Support

Static GitHub Pages site hosting the privacy policy, support page, and data-source
attribution for the **Tsubu 粒粒** JLPT Japanese vocabulary app (iOS / Android).

Live URL: https://fantasyxxj.github.io/tsubu/

## Pages

- `index.html` — support / help (简体中文 + English)
- `privacy.html` — privacy policy (English / 简体中文 / 日本語 / 繁體中文)
- `credits.html` — data sources &amp; licenses (JMdict / Tatoeba / kuromoji / Tanos / Google TTS)

## App Store / Play references

- Privacy Policy URL → `https://fantasyxxj.github.io/tsubu/privacy.html`
- Support URL → `https://fantasyxxj.github.io/tsubu/`

## Updating

Edit the HTML directly and push to `main`. GitHub Pages picks up the change within
a minute or two. Keep the privacy text in sync with the in-app policy screen.
